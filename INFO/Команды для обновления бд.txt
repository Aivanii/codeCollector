Нумерация по порядку действий.
Если номера повторяются, то не важно какой из них будет выполнен первый.
В обычную консоль:
1. указываем путь к копии бд 'cd путь'
4. mysql -u root -p 'название бд' < 'Полный путь к файлу и само его название' (Пример: C:/Users/'Имя пользователя'/Desktop/codeCollector/server/dump.sql). Это копирование бд из дампа(копии)
В sql консоль:
1. Для того, чтобы войти в sql консоль, в обычной консоли нужно ввести 'mysql -u root -p'
2. DROP DATABASE 'Название базы'; удаление бд(у нас называется Test), если делаешь в первый раз, то не нужно
3. CREATE DATABASE 'Название базы'; создание бд(у нас называется Test)


Для создания файла надо в обычную консоль ввести:
mysqldump -u root -p 'название бд, которую копируем' > 'Полный путь к файлу и его название, который создаться при выполнении команды' (Пример: C:/Users/'Имя пользователя'/Desktop/codeCollector/server/dump.sql) Это копирование бд
