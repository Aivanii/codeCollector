Синтаксис Fortran

Fortran (Formula Translation) — это один из самых ранних языков программирования, разработанный для научных и инженерных расчетов. В этой статье мы рассмотрим основные аспекты синтаксиса Fortran, включая переменные, операторы, подпрограммы и структуры управления.

Переменные и типы данных

В Fortran переменные объявляются с указанием их типа. Основные типы данных включают целые числа (INTEGER), числа с плавающей запятой (REAL), логические значения (LOGICAL) и символьные строки (CHARACTER).

1 2 3 4 5 6 7 8 9 
PROGRAM Example
    INTEGER :: age
    REAL :: height
    CHARACTER(len=20) :: name

    name = "Alice"
    age = 30
    height = 5.5
END PROGRAM Example

В этом примере мы объявили три переменные: name, age и height.

Операторы

Fortran поддерживает различные операторы, включая арифметические, логические и сравнительные операторы. Например:

1 2 
result = (5 + 3) * 2
is_greater = result > 10

В этом примере мы вычисляем результат арифметического выражения и проверяем, больше ли он 10.

Подпрограммы

Подпрограммы в Fortran могут быть определены как функции или подпрограммы. Вот пример простой функции:

1 2 3 4 5 6 
FUNCTION greet(name)
    CHARACTER(len=*), INTENT(IN) :: name
    CHARACTER(len=30) :: greet

    greet = "Привет, " // name
END FUNCTION greet

Теперь вы можете вызвать функцию greet с именем, и она вернет приветственное сообщение.

Управляющие структуры

Fortran предлагает различные конструкции управления, такие как условные операторы и циклы. Вот пример условного оператора:

1 2 3 4 5 
IF (age > 18) THEN
    PRINT *, "Взрослый"
ELSE
    PRINT *, "Несовершеннолетний"
END IF

В этом примере мы проверяем, является ли человек взрослым, и выводим соответствующее сообщение.

Заключение

Fortran остается важным языком для научных и инженерных расчетов благодаря своей простоте и эффективности. Понимание основ переменных, операторов, подпрограмм и структур управления поможет вам начать программировать на Fortran.
