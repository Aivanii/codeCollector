Синтаксис Rust

Rust — это системный язык программирования, который акцентирует внимание на безопасности памяти, производительности и параллелизме. Он идеально подходит для разработки высокопроизводительных приложений и системного программного обеспечения. В этой статье мы рассмотрим основные аспекты синтаксиса Rust, включая переменные, функции, структуры и управление потоком.

Основы синтаксиса Rust

Программа на Rust начинается с объявления функции main, которая является точкой входа в программу.

1 2 3 
fn main() {
    println!("Привет, мир!");
}

В этом примере мы создаем простую программу, которая выводит "Привет, мир!" на экран с помощью функции println!.

Переменные

Переменные в Rust по умолчанию являются неизменяемыми. Чтобы объявить изменяемую переменную, используйте ключевое слово mut.

1 2 3 4 5 
let name = "Alice"; // неизменяемая переменная
let mut age = 30; // изменяемая переменная
age += 1;

println!("Имя: {}, Возраст: {}", name, age);

В этом примере мы объявляем неизменяемую переменную name и изменяемую переменную age.

Функции

Функции в Rust определяются с помощью ключевого слова fn. Функции могут принимать параметры и возвращать значения.

1 2 3 4 5 
fn greet(name: &str) -> String {
    format!("Привет, {}!", name)
}

println!("{}", greet("Alice"));

В этом примере мы определяем функцию greet, которая принимает имя и возвращает приветственное сообщение.

Структуры

Структуры в Rust позволяют группировать данные. Они объявляются с помощью ключевого слова struct.

1 2 3 4 5 6 7 
struct Person {
    name: String,
    age: u32,
}

let alice = Person { name: String::from("Alice"), age: 30 };
println!("Имя: {}, Возраст: {}", alice.name, alice.age);

В этом примере мы создаем структуру Person и создаем экземпляр этой структуры.

Управляющие структуры

Rust поддерживает различные конструкции управления, такие как условные операторы и циклы. Вот пример условного оператора:

1 2 3 4 5 6 7 
let age = 20;

if age >= 18 {
    println!("Взрослый");
} else {
    println!("Несовершеннолетний");
}

В этом примере мы проверяем, является ли человек взрослым, и выводим соответствующее сообщение.

Заключение

Rust — это мощный язык программирования, который предлагает безопасность и производительность. Понимание основ синтаксиса, включая переменные, функции, структуры и управление потоком, поможет вам начать программировать на Rust.
